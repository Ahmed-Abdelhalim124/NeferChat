# 🩺 Nefer Guideline Chatbot

# DEMO LINK : https://drive.google.com/file/d/1ME5_8PwH0nmy8PCPVfoC0Ct_ysGIJCxG/view?usp=sharing

This project is a PDF guideline chatbot using:
- DocLayout-YOLO for layout detection
- Camelot and HuggingFace Table Transformer for table extraction
- SentenceTransformers + BM25 + FAISS for hybrid retrieval
- Groq LLM for answering questions
- Gradio for user interface
